## Quick start

1. Clone this repo using `git clone https://github.com/Sinequanonh/angularboilerplate.git`
2. `yarn install` to install dependencies
3. `gulp`

In another terminal window:
1. `cd server`
2. supervisor server.js`
Now you're ready to rumble!
